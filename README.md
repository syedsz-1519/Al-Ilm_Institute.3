# Al-Ilm Institute

A simple static project containing `al_ilm_institute.html`.

## Run locally

1. Install Python 3 if needed.
2. From this folder run:

```powershell
python -m http.server 8000
```

3. Open:

```text
http://localhost:8000/al_ilm_institute.html
```

Alternatively, run the included local server script:

```powershell
python server.py
```
